# InAmigos Foundation Awareness Webpage

This project is a responsive NGO awareness website developed using HTML and CSS.

## Features
- NGO Introduction
- Mission and Credentials
- Ongoing Projects
- Social Impact and Achievements
- Image Gallery
- Contact Information
- Social Media Links
- Call-to-Action Section

## Technologies Used
- HTML5
- CSS3

Created as part of the InAmigos Foundation Internship Program.
